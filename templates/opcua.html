► How to generate a self-signed client certificate

Run these two commands to create client_cert.der and client_key.pem. The subjectAltName URI must match the Application URI above. 

openssl req -x509 -newkey rsa:2048 \
  -keyout client_key.pem \
  -out client_cert.pem \
  -days 365 -nodes \
  -addext "subjectAltName = URI:urn:ctf:python-opcua-client"

openssl x509 -outform der -in client_cert.pem -out client_cert.der

Upload client_cert.der + client_key.pem above. The server must have your certificate in its trust list beforehand.
